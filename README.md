# Board View Keys

This mod allows you to switch to any board view (i.e. Energy, Sewers, Transport, etc.) by pressing dedicated and configurable keyboard keys.

## Configuration

Each of the options can be any key on the keyboard with an exception to some special keys like numpad keys or modifier keys.

| Option | Value Type | Default |
| ------ |:----------:| ------- |
| **Default** | `string` | `4` |
| **Energy** | `string` | `5` |
| **Sewer** | `string` | `6` |
| **Transport** | `string` | `7` |
| **Calamity** | `string` | `8` |
